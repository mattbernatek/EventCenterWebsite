# The Resonance Piano Lounge

## About the Project
The Resonance Piano Lounge is a multi-page website built HTML and CSS that highlights live events, weekly performances, and venue information for a fictional upscale piano lounge in Durham, North Carolina. 

This project demonstrates the use of semantic HTML, responsive layouts, flexbox, grid, forms, and consistent design language. 

## Website Preview
You can view the site here:
https://github.com/mattbernatek/EventCenterWebsite.git

Open index.html in your browser to view the homepage. 

## How to View
1. Clone or download this repository. 
2. Open the project folder in VS Code or your preferred editor. 
3. Open 'index.html' in a browser (or use live server functionality).
4. Navigate between pages using the top navigation bar. 

## Project Structure
- 'index.html' - Home Page
- 'about.html' - About the venue and the team
- 'events.html' - Weekly and seasonal events
- 'style.css' - Shared stylesheet for all pages
- 'images/' - Folder containing all site visuals

## Features
- Fully responsive layout using Flexbox and Grid
- Custom color palette and typography for consistent branding
- Event schedule table with hover interactivity
- Contact and reservation form with styled inputs
- Gallery section with image hover effects

## Skills demonstrated
- Semantic HTML5 structure
- CSS Flexbox and Grid layout systems
- Responsive design through media queries
- Use of custom CSS variables
- Accessibility considerations (alt text, readable contrast)
